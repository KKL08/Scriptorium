# 三档 mode 规则（review.md / direct.md 的 [[MODE_RULES]] 来源）

## proofread

低风险语文修复。
允许：修正错别字、标点、语法、语病、句子不通顺、明显衔接断裂；必要时做很小的语序调整，让句子读通。
禁止：改变语气、立场、结论、信息量、段落顺序、论证强度；不要替换作者风格，不要把口语改成正式文风。

## refine

默认自然化精修。
允许：去 AI 腔、去翻译腔、压缩冗余、减少套话、调整句式、理顺句间关系、让表达更自然。
禁止：重排大结构、替作者新增论点、删除关键限制条件、改变结论强度、把含蓄表达改成过度肯定。

## rewrite

结构和表达重写。
允许：重写句子、重组段落、改变论述顺序、强化开头和过渡，让整体更清楚有力。
必须保留：事实、结论、限定词、反对意见、风险提示、人名、数字、日期、路径、命令、API 名、函数名。
禁止：编造事实、扩大承诺、删掉 caveat、把不确定内容写成确定结论。
